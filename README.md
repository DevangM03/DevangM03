I'm a Backend & Systems Engineer based in New Delhi, completing my B.Tech in CSE (Big Data Analytics) at NSUT. I focus on building resilient distributed systems, monetizable micro-SaaS applications, and integrating agentic LLM workflows into production environments.

### ⚡ What I do well
* **Backend Architecture:** Designing multi-tenant access management systems and optimizing SQL/NoSQL databases for sub-50ms query responses.
* **AI/LLM Pipelines:** Engineering self-correcting LLM feedback loops and "Code-to-Data" workflows for financial applications.
* **High-Concurrency Systems:** Building load balancers via TCP/IP and optimizing APIs to handle concurrent user operations.
